Python + Eel: Apps de Escritorio Modernas con HTML/JS

calendar_today 04 Feb 2026 schedule 8 min person

Descubre cómo crear aplicaciones de escritorio potentes utilizando el backend de Python y la flexibilidad de HTML/CSS/JS. Guía rápida para empezar.

Descripción Técnica

Eel es una pequeña biblioteca de Python que permite crear aplicaciones de escritorio simples tipo Electron, pero mucho más ligeras. Utiliza Python para el backend y HTML/CSS/JavaScript para el frontend, combinando lo mejor del mundo web con la potencia de scripting de Python.

Fase de Explotación

Implementar Eel es sorprendentemente sencillo. A diferencia de frameworks complejos, Eel actúa como un puente directo. Aquí mostramos cómo "explotar" sus capacidades para levantar una GUI en segundos:

python
import eel

# Inicializamos la carpeta 'web' donde están nuestros archivos HTML/JS
eel.init('web')

# Exponemos una función de Python a Javascript
@eel.expose
def saludar_desde_python(x):
    return f"Hola desde Python, {x}!"

# Iniciamos la app (asume que existe un index.html en la carpeta web)
eel.start('index.html', size=(800, 600))

Mitigación

La distribución de apps Eel se puede realizar empaquetando todo con PyInstaller. Es la "mitigación" perfecta contra la dependencia de tener Python instalado en la máquina del cliente.
